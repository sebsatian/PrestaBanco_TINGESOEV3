# PrestaBanco_TINGESOEV1
Evaluación 1 Técnicas de Ingeniería de Software

1- Crear base de datos en PostgreSQL con el nombre "PrestaBancoDB"
2- Cambiar contraseña para la base de datos en /BackendPrestaBanco/src/main/resources/application.properties
3- Ejecutar el Backend
4- Ejecutar el script para poblar las tablas /BackendPrestaBanco/src/main/resources/database.sql
5- En /FrontendPrestaBanco ejecutar el comando `npm install`
6- Ejecutar el Frontend con el comando `npm run dev"
7- Ingresar a http://http://localhost:5173/
